# 153. 寻找旋转排序数组中的最小值

假设按照升序排序的数组在预先未知的某个点上进行了旋转。

( 例如，数组 [0,1,2,4,5,6,7] 可能变为 [4,5,6,7,0,1,2] )。

请找出其中最小的元素。

你可以假设数组中不存在重复元素。

示例 1:
```
输入: [3,4,5,1,2]
输出: 1
```
示例 2:
```
输入: [4,5,6,7,0,1,2]
输出: 0
```

## 解法一：二分查找
和33题基本一样。思路是二分查找时判断mid值与right值的大小。方法是二分时比较mid值与right值的大小，如果mid值大于right值，最小的值就在mid值右侧。
